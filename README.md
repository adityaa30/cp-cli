### Download Guide

```bash
# Clone the repository
git clone https://github.com/adityaa30/cpcli

# Change directory to the cloned directory
cd cpcli

# Install i
pip install -e .
``` 

### Requirements

- Python 3.6+
- [lxml](https://pypi.org/project/lxml/)
- [zope.interface](https://pypi.org/project/zope.interface/)
